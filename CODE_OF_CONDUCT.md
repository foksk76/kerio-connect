# Code Of Conduct

## Our Standard

Treat other participants with respect, patience, and curiosity.

Expected behavior:

- be constructive and specific
- assume good intent unless there is clear evidence otherwise
- make room for different experience levels
- keep technical disagreement focused on the work, not the person

Unacceptable behavior:

- harassment, abuse, or personal attacks
- insulting or dismissive language
- publishing private information without permission
- deliberately disruptive behavior in issues, pull requests, or discussions

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project communication spaces.

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates these expectations, and may limit further participation when necessary to keep the project usable and safe.

## Reporting

If conduct in this repository becomes a problem, contact the maintainer through the repository owner account on GitHub and include links or screenshots where possible.
